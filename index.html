Welcome - here you will find descriptions and demonstrations of the projects in this Github.

Wave editor - This program records and plays back sounds, and also allows viewing of a sound's waveform and frequency spectrum, as well as simple editing and filtering of the sound. Morpher - This program opens two image files and uses an image warping algorithm and cross-dissolving to make the first image appear to morph into the second. 2D vertical shooter - A one level demonstration of a vertical scrolling shooter game. Music game engine - A simple engine for a scrolling notes style music game. Music visualizer - A waveform visualizer for audio files. Motion graphics examples - Existing code for drawing 3D figures based on imported motion capture data was modified to add various visual effects that make use of the motion data.
