Project Information

Welcome - here you will find descriptions and demonstrations of my personal projects, which are contained in this Github.

Servlet/JSP course project - A Java web app using servlets and JSP that allows the user to perform basic CRUD operations on a database table of address records.

Wave editor - This program records and plays back sounds, and also allows viewing of a sound's waveform and frequency spectrum, as well as simple editing and filtering of the sound. (Java)

Morpher - This program opens two image files and uses an image warping algorithm and cross-dissolving to make the first image appear to morph into the second. (Java)

2D vertical shooter - A one level demonstration of a vertical scrolling shooter game. (C++)

Music game engine - A simple engine for a scrolling note based music game. (C++)

Music visualizer - A real-time waveform visualizer for audio files. (C++)

Motion graphics examples - Existing code for drawing 3D figures based on imported motion capture data was modified to add various visual effects that make use of the motion data. (C++)
